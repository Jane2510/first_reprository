# Задача5_10_Генератор паролей

## Описание
Данная программа генерирует случайный пароль заданной длины.
Пароль обязательно содержит:
- одну строчную букву
- одну заглавную букву
- одну цифру
- один специальный символ

Минимальная длина пароля — 12 символов.
Если передать длину меньше 12, она автоматически устанавливается равной 12.
